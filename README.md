🚀 4AT Consulting – AI-Driven Consulting Platform
📌 Overview

4AT Consulting is a next-generation consulting platform integrating AI-powered solutions with enterprise consulting services. It is designed to deliver intelligent, scalable, and efficient business solutions across multiple domains including finance, technology, and workforce transformation.

This repository contains the source code for the 4AT Consulting website and platform, built to showcase services, enable client engagement, and demonstrate AI integration capabilities.

⚙️ Installation & Setup
1. Clone the Repository
git clone https://github.com/vishnu-vardhan03/4AT-Consulting.git
cd 4AT-Consulting
2. Install Dependencies
npm install
3. Run the Development Server
npm run dev
4. Build for Production
npm run build
